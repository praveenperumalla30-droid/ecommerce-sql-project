# E-Commerce Order Management System (SQL Project)

## 📌 Project Overview
This project demonstrates the design and implementation of a relational E-commerce database using MySQL. It showcases real-world SQL skills including table creation, relationships, multi-table JOIN queries, advanced analytics, and business reporting.

## 🧱 Database Structure
The schema includes:
- Customers  
- Products  
- Orders  
- Order Items  
- Payments  

Relationships are established using foreign keys to ensure data consistency and accurate analysis.

## 💻 Features & Capabilities
- Normalized database design
- Foreign key constraints
- Multi-table `JOIN` queries
- Revenue calculation and reporting
- Customer behavior insights
- Aggregate functions (`SUM`, `COUNT`, `GROUP BY`, `HAVING`)
- Business logic queries
